# Gateway Contracts

These documents are the stable contract surfaces for the Substrate gateway boundary.

## Documents

- `operator-contract.md`: operator command family, stable wiring exports, and exit taxonomy.
- `status-schema.md`: machine-readable `substrate world gateway status --json` surface.
- `policy-evaluation.md`: fail-closed policy evaluation and secret-sourcing boundary.
- `runtime-parity.md`: typed runtime lifecycle/status authority and platform parity guarantees.
- `backend-adapter-selection.md`: backend-id selection, allowlisting, and failure taxonomy before dispatch.
- `backend-adapter-protocol.md`: adapter dispatch lifecycle, validation order, and bounded handoff rules.
- `backend-adapter-schema.md`: adopted capability/extension subset and bounded request/event/completion schema.
