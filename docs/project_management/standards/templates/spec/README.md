# Spec Templates

These templates are used by planning authors to create domain-specific spec documents selected by `spec_manifest.md`.

Standards:
- `docs/project_management/standards/PLANNING_SPEC_DETERMINATION_STANDARD.md`
- `docs/project_management/standards/PLANNING_README.md`

Templates in this directory are not auto-rendered by `make planning-new-feature` today; copy them into the feature directory and replace placeholders.
