Usage: substrate policy [OPTIONS] <COMMAND>

Commands:
  current    Policy commands scoped to the current directory/workspace (alias for `show`)
  init       Initialize <workspace_root>/.substrate/policy.yaml (creates if missing; overwrites with --force)
  show       Print the effective policy for the current directory/workspace (alias for `current show`)
  set        Update the workspace policy patch via dotted updates (alias for `workspace set`)
  global     Global policy commands ($SUBSTRATE_HOME/policy.yaml)
  workspace  Workspace-scoped policy patch commands (<workspace_root>/.substrate/policy.yaml)
  help       Print this message or the help of the given subcommand(s)

Options:
      --world     Force world isolation for this run (overrides disabled install/config/env)
      --no-world  Disable world isolation (host pass-through)
  -h, --help      Print help
